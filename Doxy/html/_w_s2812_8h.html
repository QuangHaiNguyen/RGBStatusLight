Detailed Description

This file provides the APIs to interface with WS2812 LED Strip.

The APIs abstract the underlying hardware peripheral, which generates the data stream to feed into the LED strip. The users must port the new hardware or configure the available one.

WS2812_Status WS2812_Init ( uint16_t  num_of_led,
uint16_t **  ptr 
)

Description:

This function initialize the WS2812 module. It reserves the memory for the RGB data and initializes the underlying hardware.

PRE-CONDITION: None

POST-CONDITION: Memory is reserved for RGB data, return pointer to reserve data

Parameters
num_of_ledIN number of LED we are going to use
ptrOUT pointer to the reserved memory for the RGB data
Returns
Status code

Example Example:

uint16_t *ptr_RGB_buffer
WS2812_Init(10, &ptr_RGB_buffer);
