int_fast8_t winc_adapter_spi_close ( void  )

Called by the driver to close the SPI bus. winc_adapter_spi_open should have been called before this.

Precondition
Initialize winc_adapter_init first and winc_adapter_spi_open must have been called.
Note
Implementation of this function is host dependent.
See also
winc_adapter_init
winc_adapter_spi_open
Returns
The function returns a value of 1 for successful operations and a 0 value otherwise.
