◆ HTTP_FLASH_SECTION_MAGIC

#define HTTP_FLASH_SECTION_MAGIC   0xB00B1500

Magic value to differentiate between old HTTP flash section format and newer formats. The lowest byte is ignored when checking the value as it contains the version number (it should always be 00 here, image_builder will set this value in flash).
