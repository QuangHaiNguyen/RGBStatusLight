Detailed Description

Socket accept status message structure.

Socket accept information is returned through this structure in response to the asynchronous call to the accept function. This structure together with the event SOCKET_MSG_ACCEPT are passed in parameters to the callback function.

Field Documentation

◆ sock

SOCKET sock

On a successful accept operation, the returned information is the socket ID for the accepted connection with the remote peer. Otherwise a negative error code is returned to indicate failure of the accept operation.
