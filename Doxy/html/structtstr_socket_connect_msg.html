Detailed Description

Socket connect status message structure.

Socket connect information is returned through this structure in response to the asynchronous call to the connect function. This structure together with the event SOCKET_MSG_CONNECT are passed in parameters to the callback function. If the application receives this structure with a negative value in s8Error, the application should call close.
